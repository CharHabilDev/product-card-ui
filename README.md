# Product Card UI

Modern responsive product card interface built with HTML and CSS.

## 🚀 Features

- Responsive product grid
- Interactive hover states
- Quick add-to-cart button
- Product badges (promo / discount)
- Favorite button
- Accessible semantic HTML
- CSS Grid layout
- BEM naming convention
- Mobile-first responsive design

## 🛠️ Technologies

- HTML5
- CSS3
- CSS Grid
- Flexbox

## 📚 What I Practiced

- Semantic HTML structure
- Responsive layouts with CSS Grid
- Component-based CSS architecture
- BEM methodology
- CSS variables and spacing systems
- Hover animations and transitions
- Accessibility basics (`aria-label`, visually hidden content)

## 📁 Project Structure

```text
assets/
css/
index.html
```

## 🎯 Goal

The goal of this project was to practice building a modern e-commerce UI component using only HTML and CSS before adding JavaScript interactivity later.

- Shopping cart logic
